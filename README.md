# Analyse-du-r-le-de-l-Assurance-Maladie-dans-les-d-penses-de-sant-en-France
Ce projet présente une analyse des dépenses de santé en France, en se basant sur les données de la CNS (Comptes Nationaux de la Santé – SHA). L’objectif est de visualiser et d’analyser le rôle central de l’Assurance Maladie dans le financement du système de santé français
